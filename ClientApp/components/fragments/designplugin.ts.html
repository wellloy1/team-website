Installation Instructions

Before trying to install the plugin, please make sure you have the version of Adobe Illustrator 26.2 or later.

Installing the Teamatical Art Manager Plugin Via Adobe Exchange

  1. Create an account with Teamatical here if you do not already have one. You will use these credentials to log into the Art Manager plugin during the installation process.
  2. Once you create the account please request plugin access via the feedback from.
  3. Make sure your Adobe Illustrator application is updated to the latest version and the Adobe Creative Cloud application is running on your computer.
  4. Navigate to Adobe Exchange’s page for the Teamatical Art Manager here . Make sure you are signed into the same Adobe account that you use for the Adobe Illustrator and Creative Cloud Desktop applications.
  5. On the Adobe Exchange plugin page, click “Free” button to start the download for the Teamatical plugin.
  6. The Teamatical plugin should be automatically downloaded and installed by the Creative Cloud Desktop.
  7. Restart Adobe Illustrator and proceed to the “Running the Teamatical Art Manager Plugin” section below.

ก่อนพยายามติดตั้งปลั๊กอิน โปรดตรวจสอบให้แน่ใจว่าคุณมี Adobe Illustrator 26.2 เวอร์ชันหรือใหม่กว่า

การติดตั้งปลั๊กอิน Teamatical Art Manager ผ่าน Adobe Exchange

  1. สร้างบัญชีด้วย Teamatical ที่นี่ หากคุณยังไม่มี คุณจะใช้ข้อมูลรับรองเหล่านี้เพื่อเข้าสู่ปลั๊กอิน Art Manager ในระหว่าง กระบวนการติดตั้ง
  2. ตรวจสอบให้แน่ใจว่าแอพพลิเคชั่น Adobe Illustrator ของคุณได้รับการอัพเดตเป็นเวอร์ชันล่าสุดและ Adobe Creative Cloud แอปพลิเคชันทำงานบนคอมพิวเตอร์ของคุณ
  3. ไปที่หน้าของ Adobe Exchange สำหรับ Teamatical Art Managerที่นี่ . ตรวจสอบให้แน่ใจว่าคุณได้ลงชื่อเข้าใช้บัญชี Adobe เดียวกันกับที่คุณใช้สำหรับแอพพลิเคชั่น Adobe Illustrator และ Creative Cloud Desktop
  4. ในหน้าปลั๊กอิน Adobe Exchange ให้คลิกปุ่ม“ ฟรี” เพื่อเริ่มดาวน์โหลดปลั๊กอิน Teamatical
  5. ปลั๊กอิน Teamatical ควรดาวน์โหลดและติดตั้งโดยอัตโนมัติโดย Desktop Creative Cloud
  6. รีสตาร์ท Adobe Illustrator และดำเนินการต่อในส่วน "การเรียกใช้ปลั๊กอิน Artatical Art Manager" ด้านล่าง
